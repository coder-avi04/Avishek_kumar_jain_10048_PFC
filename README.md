A simple C program to check whether a given number is a **palindrome**.  
A palindrome number is a number that remains the same when its digits are reversed.  
For example: `121`, `1331`, and `12321` are palindromes, while `123` and `4567` are not.

---
##  Features
- Takes integer input from the user.
- Checks if the number is a palindrome.
- Displays clear output messages.
- Lightweight and beginner-friendly C project.
